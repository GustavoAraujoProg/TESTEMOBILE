Guia de Navegação do Tour Virtual

Seja bem-vindo(a) ao seu tour virtual 360°! Utilize as seguintes instruções para explorar o ambiente:

Navegação entre Cômodos:

  • ⇆ Clique no **Botão de Planta** no canto esquerdo para abrir a planta baixa humanizada e selecionar um cômodo.
  • → Ou, clique nas **setas azuis flutuantes** (hotspots) no ambiente para ir diretamente para o próximo cômodo conectado.

Outros Controles:

  • ↻ Resetar Visão: Centraliza a câmera e o zoom para a visão inicial do cômodo.
  • ⎈ Capturar Tela: Salva uma imagem da sua visão atual.
  • ⬟ Tela Cheia: Ativa ou desativa o modo de tela cheia.
  • 📱 Modo VR Mobile: Ativa os sensores do celular para navegação imersiva.
  • ℹ Ajuda: Abre este guia.
  • ⌂ Início: Retorna para a tela inicial do tour.

Mais Informações

Como Usar o Tour

O tour virtual 360° é interativo e fácil de usar. Utilize as seguintes opções de navegação:

  • Clique e Arraste: Mova o mouse (ou o dedo em dispositivos móveis) para girar a visão 360°.
  • Scroll (Zoom): Use a roda do mouse ou o movimento de pinça (em dispositivos móveis) para dar zoom.
  • Setas de Navegação: Clique nas setas azuis flutuantes para ir para o próximo cômodo ou use o botão de planta no canto superior esquerdo para navegar entre os ambientes através da planta baixa.
  • Controles Inferiores: Utilize os botões na parte inferior para resetar a visão, capturar a tela, ativar/desativar tela cheia, acessar este guia de ajuda ou retornar à tela inicial.
  • Modo VR Mobile: Ative o botão 📱 para usar os sensores do seu celular e mover o dispositivo para olhar ao redor!
